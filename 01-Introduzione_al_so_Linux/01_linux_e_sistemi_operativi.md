# Linux e i Sistemi Operativi più Diffusi

## Che cos'è un Sistema Operativo?

Un **sistema operativo** (SO) è il software fondamentale che gestisce tutte le risorse hardware di un computer e fornisce i servizi di base alle applicazioni. Senza di esso, nessun programma potrebbe funzionare.

Le funzioni principali di un sistema operativo sono:

- **Gestione del processore**: assegna il tempo di CPU ai vari processi
- **Gestione della memoria**: alloca e libera la RAM per i programmi
- **Gestione dei dispositivi**: comunica con periferiche (disco, tastiera, schermo, rete)
- **Gestione del filesystem**: organizza i dati su disco in file e cartelle
- **Interfaccia utente**: fornisce la modalità di interazione (grafica o testuale)

## Il Kernel

Il cuore del sistema operativo è il **kernel** (nucleo). Si tratta del componente che opera al livello più basso del software, a diretto contatto con l'hardware. Il kernel:

- Gestisce la memoria fisica e virtuale
- Controlla i driver dei dispositivi
- Arbitrarizza l'accesso alle risorse tra i processi
- Garantisce sicurezza e isolamento tra i programmi

## I Sistemi Operativi più Diffusi

| Sistema Operativo | Tipo | Principali utenti |
|---|---|---|
| **Windows** | Proprietario (Microsoft) | Desktop aziendale e consumer |
| **macOS** | Parzialmente proprietario (Apple) | Desktop consumer, creativi |
| **Linux** | Open source | Server, sviluppatori, embedded, desktop |
| **Android** | Open source (basato su Linux) | Smartphone e tablet |
| **iOS / iPadOS** | Proprietario (Apple) | iPhone, iPad |

## Cos'è Linux?

**Linux** è un sistema operativo open source nato nel 1991 grazie a **Linus Torvalds**, uno studente universitario finlandese. Torvalds scrisse il kernel Linux ispirandosi a Unix, un sistema operativo professionale degli anni '70.

Il nome completo del sistema è tecnicamente **GNU/Linux**: il kernel Linux si combina con gli strumenti del progetto **GNU** (avviato da Richard Stallman nel 1983) per formare un sistema operativo completo.

### Caratteristiche distintive di Linux

- **Open source**: il codice sorgente è disponibile, studiabile e modificabile da chiunque
- **Gratuito**: la maggior parte delle distribuzioni non ha costi di licenza
- **Stabile e affidabile**: molti server girano per anni senza riavvii
- **Sicuro**: architettura dei permessi robusta, aggiornamenti frequenti
- **Multiutente e multitasking**: più utenti e processi simultanei
- **Portabile**: gira su architetture diverse (x86, ARM, MIPS, RISC-V...)
- **Personalizzabile**: ogni componente può essere sostituito o modificato

## Linux vs Windows vs macOS

| Caratteristica | Linux | Windows | macOS |
|---|---|---|---|
| Costo | Gratuito (solitamente) | A pagamento | Incluso con hardware Apple |
| Codice sorgente | Aperto | Chiuso | Parzialmente aperto |
| Sicurezza | Molto elevata | Media | Alta |
| Facilità d'uso | Media (migliora) | Alta | Alta |
| Personalizzazione | Massima | Limitata | Limitata |
| Uso in ambito server | Predominante (~96%) | Presente | Raro |
| Supporto hardware | Ampio | Molto ampio | Solo hardware Apple |

## Dove si usa Linux oggi?

Linux è presente ovunque, spesso in modo invisibile:

- **Server web**: oltre il 96% dei server nel mondo usa Linux (es. i server di Google, Amazon, Meta)
- **Cloud computing**: AWS, Azure, Google Cloud si basano su Linux
- **Smartphone**: Android è basato sul kernel Linux
- **Dispositivi embedded**: router, smart TV, sistemi automotive
- **Supercalcolatori**: il 100% dei 500 supercalcolatori più potenti al mondo usa Linux
- **Desktop**: crescente adozione tra sviluppatori e ambienti tecnici
