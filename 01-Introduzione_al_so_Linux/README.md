# 01 — Introduzione al Sistema Operativo Linux

## Descrizione

In questo primo modulo scopriamo cos'è Linux, da dove viene e come si inserisce nel panorama dei sistemi operativi moderni. Esploreremo il mondo dell'open source, le principali applicazioni disponibili su Linux e le diverse distribuzioni esistenti. Al termine sapremo scegliere la distribuzione più adatta alle nostre esigenze e lavorare per la prima volta con Linux.

## Obiettivi

Al termine di questo modulo sarai in grado di:

- Descrivere cos'è un sistema operativo e il ruolo di Linux
- Confrontare Linux con altri sistemi operativi diffusi (Windows, macOS)
- Spiegare il concetto di software open source e la filosofia della comunità Linux
- Elencare le principali applicazioni open source disponibili su Linux
- Distinguere le principali distribuzioni Linux e le loro caratteristiche
- Avviare Linux e orientarsi nell'ambiente di lavoro base

## Concetti Chiave

| Termine | Significato |
|---------|------------|
| **Sistema Operativo** | Software che gestisce l'hardware e fornisce servizi alle applicazioni |
| **Kernel** | Il nucleo del sistema operativo, interfaccia tra hardware e software |
| **Shell** | Interfaccia a riga di comando per interagire con il sistema |
| **Open Source** | Software il cui codice sorgente è liberamente disponibile e modificabile |
| **Distribuzione** | Un pacchetto completo con kernel Linux + software preinstallato |
| **GNU/Linux** | Il nome completo del sistema: kernel Linux + strumenti GNU |
