# 02 — Utilizzo della Command Line

## Descrizione

In questo modulo impareremo a usare la **riga di comando** (command line) di Linux in modo efficace. Scopriremo come funziona la shell Bash, come inserire e modificare i comandi, come usare le scorciatoie da tastiera per lavorare più velocemente, e soprattutto come trovare autonomamente la documentazione di qualsiasi comando.

## Obiettivi

Al termine di questo modulo sarai in grado di:

- Descrivere cos'è una shell e come funziona
- Digitare, modificare e ripetere comandi nel terminale
- Usare le scorciatoie da tastiera più utili (completamento automatico, cronologia, ecc.)
- Trovare la documentazione di qualsiasi comando con `man`, `info`, `--help` e `whatis`
- Usare le variabili speciali della shell (`$?`, `$0`, `$$`, ecc.)
- Distinguere tra comandi interni (builtin) e comandi esterni

## Concetti Chiave

| Termine | Significato |
|---------|------------|
| **Shell** | Interprete di comandi; Bash è la shell più diffusa su Linux |
| **Prompt** | Il cursore che indica che la shell è pronta a ricevere comandi |
| **Comando** | Un programma o builtin che la shell può eseguire |
| **Argomento** | Un valore passato al comando (es. un nome di file) |
| **Opzione/Flag** | Modifica il comportamento del comando (es. `-l`, `--help`) |
| **Exit status** | Codice numerico restituito da un comando (0 = successo) |
| **man page** | La pagina del manuale integrato di Linux |
