# 04 — Visualizzazione e Ricerca nei File

## Descrizione

In questo modulo impareremo a leggere e ispezionare il contenuto dei file, a modificarli con l'editor `nano`, a comprimerli con `gzip` e `tar`, e a cercare dati al loro interno con strumenti potenti come `grep`, `sort`, `cut` e `wc`. Sono le competenze fondamentali per lavorare con file di log, dati e configurazioni.

## Obiettivi

Al termine di questo modulo sarai in grado di:

- Visualizzare il contenuto di file con `cat`, `less`, `head`, `tail`, `more`
- Modificare file di testo con l'editor `nano`
- Creare e decomprimere archivi con `tar`, `gzip`, `bzip2`, `zip`
- Cercare testo nei file con `grep` e le espressioni regolari di base
- Filtrare, ordinare ed estrarre dati con `sort`, `cut`, `uniq`, `wc`
